## Project Overview
This repository contains my Digital Literacy Portfolio developed as part of the CSE0001 course. As a Student Digital Ambassador, the objective of this project is to demonstrate an understanding of essential digital skills, including online safety, professional communication, and building a strong digital presence.

The project is divided into five modules, each focusing on a key aspect of digital literacy. Through this portfolio, I have applied practical tools and concepts to create resources that can help students navigate the digital world responsibly and effectively.
